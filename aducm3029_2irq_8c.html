◆ irq_source_enable()

int32_t irq_source_enable ( struct irq_ctrl_desc *  desc,
uint32_t  irq_id 
)

Enable the interrupt.

In order for the irq_id interrupt to be triggered the GPIO associated pin must be set as input.
The associated values are:

  • ID 0 -> GPIO15
  • ID 1 -> GPIO16
  • ID 2 -> GPIO13
  • ID 3 -> GPIO33
Parameters
desc- Interrupt controller descriptor.
irq_id- Id of the interrupt
Returns
SUCCESS in case of success, FAILURE otherwise.
